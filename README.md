# Wave Function Collapse

An algorithm for procedurally generating tile maps.

## Run

```bash
mvn exec:java
```

## Keybindings

| Key   | Action |
|-------|--------|
| ESC   | Quit   |
| SPACE | Pause  |
| S     | Step   |
| F     | Finish |
| R     | Reset  |
